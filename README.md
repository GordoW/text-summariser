# Text Summariser API

A simple text summarisation API built using Python and FastAPI.

This project is my first step into building AI-powered applications, focusing on backend API development and preparing for integration with large language models (LLMs).

## Features

- REST API built with FastAPI
- Accepts text input through an API endpoint
- Returns processed text output
- Automatic API documentation using Swagger UI

## Technologies Used

- Python
- FastAPI
- Uvicorn
- Pydantic
